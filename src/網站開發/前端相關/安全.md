# 前端安全學習筆記

## 同源政策 (Same-origin policy)

同源政策限制了程式碼和不同網域資源間的互動。

### 跨域方案

* 使用 使用CORS允許跨來源存取
* 透過 document.domain 跨域 (將被禁用)
